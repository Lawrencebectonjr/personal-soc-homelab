# Personal SOC Home Lab

## Overview
This project documents the design, deployment, and hardening of a personal Security Operations Center (SOC) home lab using consumer hardware and open-source tools.

The goal is to demonstrate foundational SOC skills, including network visibility, host hardening, firewall configuration, logging, and attack surface reduction.

This is a personal learning project focused on:
- Linux administration
- Networking fundamentals
- Security monitoring
- Log analysis
- Detection engineering basics

## Lab Environment
- Ubuntu Server (Primary monitored host)
- HP Laptop (Attacker / Scanner / Analyst workstation)
- TP-Link Home Router
- UFW (Host-based firewall)
- Nmap (Network scanning & validation)
